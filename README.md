<summary><h2>Language Features</h2></summary>

## 1. Auto Keyword: Automatic type deduction.
```
auto x = 5;  // x is int
```
## 2. Range-Based For Loop: Simplified iteration over collections.
```
int arr[] = {1, 2, 3, 4, 5};

for (int value : arr)
    std::cout << value;
```
